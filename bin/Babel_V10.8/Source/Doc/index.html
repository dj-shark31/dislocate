Babel package

version: 10.5 (May 2020)

Programs:
    - babel: creates line defects and calculate elastic energy
    - displacement: calculates displacement, strain, Nye tensor on each atom between an input and a reference structure
    - patternInit and patternDetect: defines reference crystallographic patterns in order to calculate then Nye tensors with program displacement
    - prepareDrag: interpolates between an initial and a final configuration and build constraint file so as to to run then constrained minimization (drag)
    - displacementFit (not documented yet): determines the dislocation properties (positions and Burgers vectors) by a least-square fitting of the displacement field extracted from atomic simulations with program displacement
    - vitekFit (not documented yet): determines the dislocation properties, positions and Burgers vectors, by a least-square fitting of the Vitek map, i.e differential displacements extracted from atomic simulations with programs displacement and vitek (vitek program in subdirectory Source/Tools).
